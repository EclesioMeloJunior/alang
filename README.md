# Monkey Language

Writing a interpreter to the turing complete Monkey Language in Go.
_Following the book: Writing an interpreter in Go_

## Examples

```
let add = fn(x,y) {
    return x+y;
};

let first_num = 5;
let second_num = 10;

let result = add(first_num, second_num);
```
